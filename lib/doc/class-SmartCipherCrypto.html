Class SmartCipherCrypto

Class: SmartCipherCrypto Provides a built-in based feature to handle the Blowfish (CBC) encryption / decryption. This provides an advanced crypto handler for Blowfish CBC algorithm.

// Usage example:
SmartCipherCrypto::some_method_of_this_class(...);
Final
Package: Crypto
Version: v.180124
Depends: classes: Smart
Hints: Blowfish is a 64-bit (8 bytes) block cipher. Max Key is up to 56 chars length (56 bytes = 448 bits). The CBC mode requires a initialization vector (iv).
Usage: static object: Class::method() - This class provides only STATIC methods
