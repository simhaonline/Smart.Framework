Class SmartMailerMimeDecode

Class: SmartMailerMimeDecode - provides an eMail MIME decoder. This class is for very advanced use.

It just implements the mime decoding to a PHP array by decoding Mime Email Messages. To easy parse mime messages and display them on-the-fly use the class: SmartMailerMimeParser.
