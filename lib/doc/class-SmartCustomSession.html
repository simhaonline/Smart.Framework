Class SmartCustomSession

Class App.Custom.Session.Redis - Provides a custom session adapter to use Redis (an alternative for the default files based session). NOTICE: using this adapter if the Session is set to expire as 0 (when browser is closed), in redis the session will expire at session.gc_maxlifetime seconds ...

To use your own custom adapter for the session in Smart.Framework you have to build it by extending the SmartAbstractCustomSession abstract class and define it in etc/init.php at the begining such as: define('SMART_FRAMEWORK_SESSION_CUSTOM_HANDLER', 'modules/app/session-custom-adapter.php');
