Package Application

Classes summary
SmartAbstractAppController Class: SmartAbstractAppController - Abstract Application Controller, provides the Abstract Definitions to create controllers in modules.
SmartAdapterTextTranslations Class App.Custom.TextTranslationsAdapter.Yaml - YAML files based text translations adapter (default).
SmartAppInfo Class: SmartAppInfo
SmartAuth Class: Smart Authentication - provides a safe, in-memory object, to store the authentication data and to provide a standard way to work with authentication inside the smartframework / modules.
SmartCache Class: SmartCache - Provides per Execution, Volatille Cache (in-PHP-Memory volatille cache).
SmartCustomSession Class App.Custom.Session.Redis - Provides a custom session adapter to use Redis (an alternative for the default files based session). NOTICE: using this adapter if the Session is set to expire as 0 (when browser is closed), in redis the session will expire at session.gc_maxlifetime seconds ...
SmartFrameworkSecurity Class Smart.Framework Security It may be used anywhere inside Smart.Framework or by Plugins and Application Modules.
SmartPersistentCache Class: App.Custom.PersistentCacheAdapter.Redis adapter based on Redis - provides a persistent Cache (in-Redis-Memory), that can be shared and/or reused between multiple PHP executions. If Redis is not available it will be replaced by the Blackhole Persistent Cache adapter that will provide the compatibility adapter for the case there is no real Persistent Cache available.
SmartSession Class: SmartSession - provides an Application Session Container.
SmartTextTranslations Class Smart Text Translations. It provides a Language Based Text Translations Layer for the Smart.Framework based Applications.
