Class SmartMailerSmtpClient

Class: SmartMailerSmtpClient - provides the raw communication protocol between PHP and a SMTP server with support for SSL/TLS. This class is for very advanced use.

It just implements the communication protocol between PHP and a SMTP server. It does and NOT implement the Mail Send Client. To easy send email messages use: SmartMailerUtils::send_email() / SmartMailerUtils::send_extended_email() functions. For more advanced needs on send emails use the SmartMailerSend class ...
